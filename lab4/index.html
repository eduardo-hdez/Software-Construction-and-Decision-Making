Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias

  • Java es un lenguaje orientado a objetos puros
  • Java es lenguaje compilado y JavaScript interpretado
  • Java es un tipado débil y JavaScript un tipado débil
  • Java se ejecuta en una máquina virtual y JavaScript en el navegador
  • Java es un lenguaje estático y JavaScript es un lenguaje dinámico

Semejanzas

  • Los dos son lenguajes orientados a objetos
  • El manejo de excepciones es similar
  • Su sintaxis es un poco similar

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • Date.getDay(): Devuelve el día de la semana (0-6). Domingo = 0.
  • Date.getDate(): Devuelve el día del mes (1-31).
  • Date.getHours(): Devuelve la hora (0-23).
  • Date.getMonth(): Devuelve el mes. Empieza en 0. Enero = 0, Febrero = 1, etc.
  • Date.getFullYear(): Devuelve el año en formato de 4 dígitos.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • push(element): Agrega un elemento al final del arreglo. Modifica el arreglo original.
  • pop(): Elimina el último elemento y lo devuelve.
  • shift(): Elimina el primer elemento.
  • unshift(): Agrega un elemento al inicio.
  • filter(callback): Devuelve un nuevo arreglo con los elementos que cumplen una condición.

¿Cómo se declara una variable con alcance local dentro de una función?

En JS, una variable tiene alcance local cuando solo existe dentro de la función donde fue declarada. Si se declara fuera una función se considera como global.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Que esas variables serán usadas por varias funciones o otras partes del código.
